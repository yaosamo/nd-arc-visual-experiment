# ND ARC Visual Experiment

Standalone visual identity experiment for a Navigation Display (ARC mode, airliner EFIS style).

## Local run

From this folder:

```bash
python3 -m http.server 4173
```

Open:

- `http://localhost:4173/`

## Notes

- Single SVG root (`640x720`)
- High-contrast avionics styling (black / white / phosphor green)
- Vanilla HTML/CSS/JS only
